## 섹션 2. Django의 기본 구조
### 간단한 웹 페이지 띄우기
* `views.py` : 웹 화면에 어떤 데이터를 보여줄 것인지를 정의하는 파일
* `urls.py` : 웹 사이트의 url과 각 웹 화면(페이지)를 연결(매핑) 정의하는 파일. 한 사이트의 도메인 이하 url을 웹앱의 path라고도 하는데 이렇게 어떤 화면으로 연결될지 정의하고 연결해주는 것을 라우팅이라고 한다. (route)
